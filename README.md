## How to Install

1. Download the latest version of Swift Publisher for macOS.  
2. Open the DMG installer and move SwiftPublisher.app to the Applications folder.  
3. Launch the app and allow it through macOS Gatekeeper if required.

## Major Features
- Templates for brochures, flyers, newsletters, and more  
- Extensive library of graphic elements and fonts  
- Advanced photo editing tools for professional results  
- Multiple layer support for complex designs  
- Simple drag-and-drop design interface for fast creation  
- Customizable text, image, and shape properties  
- PDF, JPEG, and PNG export options for sharing  
- Full compatibility with macOS's drag-and-drop capabilities  
- Ability to print directly from the app or save as high-quality PDFs  
- Built-in spell checker and typography tools  
- Integration with iCloud for seamless multi-device use  
- Video tutorials to help users get started quickly
